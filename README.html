### ¿Qué es CSS Styling Images? Para empezar, CSS Styling Images, como su propio nombre lo indica, es la forma que tiene el lenguaje CSS para estilizar imagenes a tu antojo, pudiendo asi cambiar su posición, tamaño, forma, bordes, entre otras muchas cosas. Esto se hace en CSS ya que este lenguaje se caracteriza por su capacidad de presentar y estilizar una página como el programador desee. Aquí dejaré un ejemplo de cómo quedaría una imagen estilizada:
